## Technical Highlights
* Detected if uploaded pictures contains animals by using `Core ML` framework.
* Used `Firebase Storage` to store and manage data.
* Experienced with `Auto Layout` to develop applications for multiple screen size using different constraints.
* Experienced in `MVC` design architecture.
* Worked on `RESTful API` web service calls to download shelter information.
* Worked on fetching data from web services and parsing data by `JSON`.
* Used `Core Data` to save favorite animal collection.
* Implemented MapView to display the shelter location using `MapKit`.
* Designed the UI and user flow.
* Installed frameworks and tracked dependencies using `CocoaPods` to manage libraries.
* Used `AVFoundation` to handle audio playback and video streaming.
* Followed the `Agile` methodology for implementing the project.
